# GovDB Shield - 政府IT系统数据库访问控制

专为政府IT管理员设计的零信任数据库访问控制系统，用于防止离职员工和未授权访问敏感数据。

## 核心功能

### 📊 系统概览
- 实时安全状态监控
- 关键指标仪表盘
- 安全事件时间线
- 离职员工访问尝试拦截演示

### 🔐 访问控制
- 角色权限管理
- 访问策略配置
- 敏感操作审批流程
- 权限变更历史

### 👁 实时监控
- 数据库健康状态
- 活跃会话监控
- 实时访问活动流
- 性能指标追踪

### 🚨 异常警报
- 多级别风险分类
- 警报状态管理
- 响应操作建议
- 处理历史记录

### 📋 审计日志
- 完整活动记录
- 高级搜索筛选
- 合规管理工具包
- 多法规支持（等保2.0、GDPR、HIPAA、PCI）

## 技术栈

- **框架**: Next.js 16.2.4 (App Router)
- **UI**: React 19 + Tailwind CSS 4
- **语言**: TypeScript (严格模式)
- **构建**: Turbopack
- **代码质量**: ESLint, Vitest
- **包管理**: pnpm

## 快速开始

```bash
# 安装依赖
pnpm install

# 开发模式
pnpm dev

# 生产构建
pnpm build

# 启动生产服务器
pnpm start

# 代码检查
pnpm lint

# 运行测试
pnpm test
```

## 设计原则

- **零信任架构**: 永不信任，始终验证
- **最小权限原则**: 仅授予必要权限
- **完整审计追踪**: 所有操作可追溯
- **合规优先**: 符合多项安全法规要求
